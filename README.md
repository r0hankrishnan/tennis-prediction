# tennis-prediction
Using Tennis Abstract data to explore predicting tennis matches


Data sources:
- [Jeff Sackman Tennis Match Data](https://github.com/JeffSackmann/tennis_atp)
  - Repo with CSVs of matches up until 2024
  - Several missing data issues
  - No player id
- [TML Database](https://github.com/Tennismylife/TML-Database)
  - More complete
  - Newer
  - Matches up until 2026
  - Seems to be updated regularly
